## 概要

配列 $S, T$ に対し，最長共通部分列 (LCS: Longest Common Subsequence) を求める．
最長共通部分列とは，2つの配列において，双方に現れる部分列の中で最長のものを指す．

アルゴリズムの計算量は $\mathcal{O}(\lvert S \rvert \lvert T \rvert)$ となる．
