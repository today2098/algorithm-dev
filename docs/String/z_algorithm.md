## 概要

配列 $S$ に対して，$S$ と各 $i$ における $S[i:]$ の最長共通接頭辞 (LCP: Longest Common Prefix) の長さを求める．

アルゴリズムの計算量は $\mathcal{O}(\lvert S \rvert)$ となる．
